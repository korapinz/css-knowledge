CSS & Selector คืออะไร

selector { property : value; property:value... }

Selector คือ แท็ก HTML, ID และ Class ที่ต้องกำหนดคุณสมบัติ แบ่งออกเป็น 3 รูปแบบ

Element Selector คือ การกำหนด style ให้แท็ก HTML โดยตรง

Class Selector คือ การกำหนด Style โดยใช้ชื่อ Class เป็น Selector โดยใช้เครื่องหมายจุด (.) สามารถมีค่าซ้ำกันได้

ID Selector คือ การกำหนด style โดยระบุชื่อ ID เป็นรหัสเฉพาะของแท็กหลังเครื่องหมาย # โดย ID ไม่สามารถกำหนดค่าซ้ำกันได้

Declaration ใช้สำหรับกำหนดค่าให้กับ Selector ซึ่งมีส่วนประกอบอยู่ด้วยกัน 2 ส่วน คือ

Property หรือ Attribute คือ คุณสมบัติที่ต้องการกำหนดให้กับ Selector

Value คือ ค่าที่กำหนดให้กับ Property หรือ Attribute

รูปแบบการประกาศใช้ CSS

Internal

css อยู่ไฟล์เดียวกัน มี Inline (เขียนลงใน tag นั้นเลย) กับ Embeded (เขียนแบบใช้ selector)

External

แยกเป็นอีกไฟล์โดยมีนามสกุล .css เช่น style.css และทำการ import เข้ามาในไฟล์ html ของเราโดยการใช้ tag link (link rel='stylesheet' href="style.css")"

การกำหนดสี ( color )

กำหนดได้ 4 รูปแบบ

1. แบบปกติ color: red;

2. แบบ RGB rgb: rgb(red,green,blue), rgb(30%,20%,45%)

- rgb แบบมีความโปร่งใส rgba: rgb(red,green,blue,alpha), rgba(255,2,50,100)

3. ฐาน 16 Hexadecimal: #000 #FFF #EE82EE

4. HSL ( hue,saturation,lightness ): color:hsl(27,100%,50%) ค่าความสดใสของสี ค่าความเข้มข้นของสี

การกำหนดขนาดข้อความ ( font-size )

กำหนดได้ 2 รูปแบบ

1. แบบค่าคงที่: xx-small, x-small, medium, large, x-large, xx-large

2. แบบตัวเลข: 10px 20px ...

การกำหนดความหนาข้อความ ( font-weight )

กำหนดได้ 2 รูปแบบ

1. แบบค่าคงที่: lighter, bold, bolder, normal

2. แบบตัวเลข: 100, 200, ..., 800, 900

การกำหนดลักษณะข้อความ ( text-decoration )

กำหนดได้ 4 รูปแบบ

1. none: ไม่ขีดเส้น

2. underline: ขีดเส้นใต้

3. overline: ขีดเส้นเหนือข้อความ

4. line-through: ขีดเส้นทับข้อความ

การจัดแนวข้อความ ( text-align )

กำหนดได้ 4 รูปแบบ

1. left: ชิดซ้าย

2. right: ชิดขวา

3. center: อยู่ตรงกลาง

4. justify: กระจายข้อความยาวๆ ให้ทุกบรรทัดเท่ากัน

การกำหนดความกว้างและความสูง ( width & height )

auto คือ browser จะกำหนดให้เอง

length คือ ความสูง/ความกล้างแบบหน่วยวัด เช่น px cm เป็นต้น

% คือ ความกว้างแบบ % ต่อหน้าจอ ( ยืดหยุ่น )

เราสามารถกำหนดให้มีค่า max/min ของขอบเขตของค่านั้นๆได้ด้วยการใช้ max/min เช่น max-width: 200px

การกำหนดเส้นขอบแบบภาพรวม ( border: <รูปแบบเส้น> <ขนาด> <สี> )

รูปแบบเส้น: solid เส้นทึบ, dotted จุด, dashed เส้นปะ, double เส้นคู่

ขนาด: 10px

สี: blck

กำหนด style เส้นขอบ ( border-style: ชื่อเส้นขอบ )

กำหนด color เส้นขอบ ( border-color: สี )

กำหนด size เส้นขอบ ( border-width: ขนาด )

กำหนด radius เส้นขอบ ( border-radius: ค่าความโค้ง มุมบนซ้าย, มุมบนขวา, มุมล่างขวา, มุมล่างซ้าย )

การกำหนด style ให้กับ Link ( a:link )

a:link กำหนด style ให้ลิงค์ที่ยังไม่ถูกคลิก

Google

a:hover กำหนด style ให้ลิงค์เมื่อนำเมาส์ไปวาง

Facebook

a:visited กำหนด style ให้ลิงค์ที่ถูกคลิกแล้ว

Youtube

a:active กำหนด style ให้ลิงค์ขณะถูกคลิก

การจัดตำแหน่งด้วย Position

static คือ default

relative คือ จัดวางและกำหนดตำแหน่งวัตถุโดยการนับจากจุดที่วัตถุนั้นๆอยู่ (ง่ายๆ ก็คือ ปรับให้ออกจากตำแหน่งเดิม)

absolute คือ จัดวางวัตถุให้ไปอยู่ตำแหน่งใดก็ได้ แต่ต้องกำหนดตำแหน่งจากขอบ element ที่บรรจุ วัตถุนั้น ๆ อีกที ถ้าไม่มีอะไรครอบก็ถือว่า body เป็น element ที่ครอบวัตถุที่ระบุ absolute

fixed ใช้ในการจัดวางวัตถุให้อยู่ตำแหน่งเดิม ไม่ว่าจะเลื่อนไปไหน มันก็จะสิงอยู่ตรงนั้นล่ะ เราก็จะเห็นมันตลอด

sticky ให้วัตถุติดขอบเมื่อเลื่อนไปถึง แบบเมื่อเลื่อนไปเจอมัน มันก็จะติดขอบไม่ไปไหนทั้งนั้น

Overflow จัดการเนื้อหาที่อยู่ใน div

overflow: visible ไม่ทำไร

overflow: hidden ซ่อนเนื้อหาที่ล้น

overflow: scroll ดูเนื้อหาที่เกินโดยการ scroll (overflow-y: scroll, overflow-x: scroll;)
