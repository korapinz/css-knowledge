CSS & Selector คืออะไร

Selector คือ แท็ก HTML, ID และ Class ที่ต้องกำหนดคุณสมบัติ แบ่งออกเป็น 3 รูปแบบ

Element Selector คือ การกำหนด style ให้แท็ก HTML โดยตรง

Class Selector คือ การกำหนด Style โดยใช้ชื่อ Class เป็น Selector โดยใช้เครื่องหมายจุด (.) สามารถมีค่าซ้ำกันได้

ID Selector คือ การกำหนด style โดยระบุชื่อ ID เป็นรหัสเฉพาะของแท็กหลังเครื่องหมาย # โดย ID ไม่สามารถกำหนดค่าซ้ำกันได้

Declaration ใช้สำหรับกำหนดค่าให้กับ Selector ซึ่งมีส่วนประกอบอยู่ด้วยกัน 2 ส่วน คือ

Property หรือ Attribute คือ คุณสมบัติที่ต้องการกำหนดให้กับ Selector

Value คือ ค่าที่กำหนดให้กับ Property หรือ Attribute

selector { property : value; property:value... }

รูปแบบการประกาศใช้ CSS

Internal

css อยู่ไฟล์เดียวกัน มี Inline (เขียนลงใน tag นั้นเลย) กับ Embeded (เขียนแบบใช้ selector)

External

แยกเป็นอีกไฟล์โดยมีนามสกุล .css เช่น style.css และทำการ import เข้ามาในไฟล์ html ของเราโดยการใช้ tag link (link rel='stylesheet' href="style.css")"

การกำหนดสี

color กำหนดได้ 4 รูปแบบ

1.แบบปกติ color: red;

2.แบบ RGB rgb: rgb(red,green,blue), rgb(30%,20%,45%)

- rgb แบบมีความโปร่งใส rgba: rgb(red,green,blue,alpha), rgba(255,2,50,100)

3.ฐาน 16 Hexadecimal: #000 #FFF #EE82EE

4.HSL ( hue,saturation,lightness ): color:hsl(27,100%,50%) ค่าความสดใสของสี ค่าความเข้มข้นของสี
